API ECOMMERCE

Bienvenido a la API JSON para E-commerce, diseñada para proporcionar acceso eficiente a la información crucial de una tienda en línea desarrollada como un Desafío Front End.

Esta API permite interactuar con productos, categorías, subcategorías y usuarios, facilitando la gestión de una plataforma de comercio electrónico.

PRODUCTOS

Los productos son los elementos que se venden en la tienda en línea.

Se puede obtener el listado de todos los productos.

GET https://node-mongo-ecommerce.onrender.com/api/productos

Se pueden traer los productos por ID.

GET https://node-mongo-ecommerce.onrender.com/api/productos/:id

CATEGORÍAS

Se puede obtener el listado de todas las categorías.

GET https://node-mongo-ecommerce.onrender.com/api/categorias

Se pueden traer las categorías por ID.

GET https://node-mongo-ecommerce.onrender.com/api/categorias/:id

SUBCATEGORÍAS

Se puede obtener el listado de todas las subcategorías.

GET https://node-mongo-ecommerce.onrender.com/api/subcategorias

Se pueden traer los subcategorías por ID.

GET https://node-mongo-ecommerce.onrender.com/api/subcategorias/:id

BUSCAR

Se puede obtener un listado de productos que coincidan con el nombre-termino.

GET https://node-mongo-ecommerce.onrender.com/api/buscar/:nombre

SESIÓN

Se inicia sesión ingresando email y contraseña.

GET https://node-mongo-ecommerce.onrender.com/api/auth/login 
{
"email": "xxxxxx@xxxx.com",
"contraseña": xxxxxxx
}

Se puede crear una cuenta ingresando los datos solicitados.

POST https://node-mongo-ecommerce.onrender.com/api/auth/ 
{
"email": "xxxxxx@xxxx.com",
